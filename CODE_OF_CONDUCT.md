# Code of Conduct

## Our Commitment

We want Bank Support to be a respectful, safe, and productive place for communication.

This repository is intended for:
- bug reports
- feature requests
- support questions
- access and billing discussions
- constructive feedback

By participating in this repository, you agree to communicate respectfully and professionally.

---

## Expected Behavior

Please:

- be respectful
- be constructive
- be patient
- focus on facts
- describe problems clearly
- provide helpful context
- accept that not every request will be implemented

---

## Unacceptable Behavior

The following behavior is not allowed:

- harassment
- insults
- threats
- aggressive or abusive language
- discrimination of any kind
- spam
- trolling
- intentional misinformation
- publishing sensitive or private information
- sharing passwords, tokens, payment data, or internal credentials

---

## Scope

This Code of Conduct applies to:
- Issues
- Discussions
- Pull Requests
- comments and other interactions in this repository

---

## Moderation

Repository maintainers may:

- edit or remove inappropriate content
- close off-topic or abusive discussions
- mark spam or low-quality issues
- limit interactions if necessary
- block users in serious cases

---

## Security and Sensitive Information

Do not post:
- passwords
- tokens
- secret keys
- private URLs
- personal data
- payment details
- internal system credentials

If your concern is security-related, do not post it publicly. Contact the project owner directly.

---

## Final Note

Constructive communication helps resolve issues faster and makes the platform better for everyone.

Thank you for helping keep Bank Support useful and respectful.
